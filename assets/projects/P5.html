Autodocking of Mobile Robot

Dec 2021 - Jul 2022

Project Description:
Project for my undergraduate research opportunity project (UROP) conducted in partnership with the robotics start-up company Solustar Pte Ltd. Here, I designed an auto-docking solution for the mobile robot to dock with the charging station, thus eliminating the need for human intervention.

In this project, I utilized fiducial marker (AR Tag) detection for the robot to localize itself. Then, I implemented a PI-control system to adjust the differential drive mobile robot kinematics. The robot will adjust its linear and angular velocity accordingly. For detailed implementation, refer to my GitHub or report.

Knowledge Demonstrated:
Motion Control, PID, Marker Detection

Skillset Demonstrated:
ROS, Python, PID, Odometry, RVIZ, Gazebo
