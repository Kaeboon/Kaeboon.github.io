Augmented Reality Marker Detection

2022

Project Description:
This personal project aims to design a marker-based AR application and website to track images and spawn a 3D model on top of it.

I used Vuforia on Unity to build the project. Packages used include ARFoundation, ARCore, ARKit.

Knowledge Demonstrated:
Augmented Reality

Skillset Demonstrated:
