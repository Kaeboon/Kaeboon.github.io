Autonomous TurtleBot Navigation

Jan 2024 - Present

Project Description:
Project for EE4308: Autonomous Robotic System. This project aims to implement navigation software for the Drone to navigate towards placed waypoints.

In this project, few Drone sensors measurements are utilized, including GPS, Magnetometer, Sonar and Barometer. An extended Kalman Filter (EKF) are used to estimate the pose and speed of the flying drone. The goal is to let the drone travel between different waypoints while a turtlebot navigates an obstacle course below.

Knowledge Demonstrated:
Extended Kalman Filter

Skillset Demonstrated:
ROS, C++, Odometry, Occupancy Grid, Inflation, Inverse Sensor Model, Line algorithms, Path Planners (A*STAR, Theta* etc.), Motion Control, Simple Pursuit, Spline Trajectories, PID Motion Control, Extended Kalman Filter, etc.
