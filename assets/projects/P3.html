Quadruped Locomotion using Deep Reinforcement Learning

Aug 2023 - Dec 2023

Project Description:
Project for ME5418: Machine Learning in Robotics. This project aims to train a quadruped robot for forward motion utilizing deep reinforcement learning algorithms.

For this project, I applied the Deep Deterministic Policy Gradient Algorithm to train our policy. The reason for choosing DDPG is the continuous state values we are dealing with in our environment. The state space employed in this project includes robot position, velocity, leg interactions with the ground, etc. I have employed a simple reward structure here, which focuses solely on rewarding forward velocity and penalizing lateral movements.

For this project, I coded the OpenAI simulation, neural network, and the reinforcement learning agent from scratch. For further implementation details, you may refer to my report attached below.
